# David Han's Portfolio

## Description

My Portfolio site created with GatsbyJS using Contentful as the Headless CMS

Demo: https://david-han.netlify.app/

#### Technologies

- GatsbyJs
- React
- Contentful
- SCSS

#### How to Setup and Run 

```html
  // Local build at http://localhost:8000/
  gatsby develop

  // Production build
  gatsby build
```
